# UX e acessibilidade

Projeto simples e acessível para apresentar conceitos de UX e inclusão digital em uma interface moderna.

## O que foi desenvolvido
- Página inicial com foco em acessibilidade e experiência do usuário.
- Estrutura semântica com navegação clara e conteúdo organizado.
- Estilos com contraste adequado, foco visível e layout responsivo.
- Formulário com validação básica e mensagem de confirmação acessível.

## Como visualizar
1. Abra o arquivo [index.html](index.html) em um navegador.
2. Ou execute um servidor local na pasta do projeto, por exemplo:
   - `python3 -m http.server 8000`
   - Acesse `http://localhost:8000`.
